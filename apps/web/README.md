# Web App

React + Vite frontend for the collaborative document editor.

## Scripts

- `pnpm dev`: start the web app in development mode
- `pnpm build`: type-check and build the production bundle
- `pnpm typecheck`: run the TypeScript compiler without emitting files
- `pnpm lint`: run the configured ESLint checks
- `pnpm test`: run the web test suite

## Notes

- The app expects the shared contracts package to be built in the monorepo workspace.
- Runtime configuration lives in `.env` files documented by `.env.example`.
